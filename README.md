# Machine Learning Algorithms & Implementations 🚀

This repository contains a structured collection of Machine Learning algorithms I implemented from scratch and applied to real-world datasets during my AI/ML Bootcamp. 

The goal of this repository is to demonstrate a deep understanding of core ML mathematics, data preprocessing, and model evaluation techniques across both Supervised and Unsupervised learning.

## 📂 Repository Structure

### 1. Supervised Learning (`/supervised_learning`)
* **Linear Models:** Linear Regression, Logistic Regression, Ridge, and Lasso.
* **Tree & Ensemble Methods:** Decision Trees, Random Forest, and AdaBoost.
* **Other Classifiers:** Support Vector Machines (SVM/SVC), K-Nearest Neighbors (KNN), and Naive Bayes.

### 2. Unsupervised Learning (`/unsupervised_learning`)
* **Clustering:** K-Means Clustering, Hierarchical Clustering, and DBSCAN.
* **Dimensionality Reduction:** Principal Component Analysis (PCA).
* **Anomaly Detection:** Implementation of anomaly detection techniques.

### 3. Datasets (`/datasets`)
Contains the cleaned datasets used to train and evaluate the models, including:
* Algerian Forest Fires
* Healthcare Dataset
* CarDekho Pricing Data

### 4. Saved Models (`/saved_models`)
Contains serialized models (`.pkl` files) ready for deployment (e.g., standard scalers, ridge regressors).

## 🛠️ Tools & Technologies Used
* **Languages:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook

## 💡 Key Learnings
Through building these models, I gained hands-on experience with:
* Handling missing data and outlier detection.
* Feature scaling (Standardization/Normalization) and Feature Engineering.
* Hyperparameter tuning to prevent overfitting/underfitting.
* Evaluating models using metrics like RMSE, R2 Score, Accuracy, and AUC/ROC.
